# ud120-projects

Starter project code for students taking Udacity ud120

## Requirements

* Python 2.6 or 2.7.
* Computer with at least 2GB of RAM

### Packages

    pip install nltk
    pip install numpy
    pip install scipy
    pip install scikit-learn

## Setup

Run

    cd tools # startup script assumes this is your working directory
    ./startup.py
